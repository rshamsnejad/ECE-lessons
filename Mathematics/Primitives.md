# Primitives

## Common functions' primitives

#### Constant

$ \forall (\lambda,C) \in \R^2 $
$ \boxed{\int(\lambda)dx = \lambda \times x + C} $

#### Polynoms

$ \forall  (x,C) \in \R^2, \forall n \in \N^* $
$ \boxed{\int(x^n)dx = \frac{x^{(n+1)}}{n+1} + C} $

This gives in particular :

* $ \boxed{\int(x)dx}
\\ = \int(x^1)dx
\\ = \frac{x^{(1+1)}}{1+1} + C
\\ \boxed{= \frac{x^2}{2} + C} $
* $ \boxed{\int(x^2)dx}
\\ = \frac{x^{(2+1)}}{2+1} + C
\\ \boxed{= \frac{x^3}{3} + C} $
